The purpose of this project is to rewrite the original `mem` application (referred to as "mem-nu")
that is already available in your shell. We are rewriting it in Rust with a modified API.

Load the `mem` skilll and follow the conventions.

Read master mem document in `.mem/master/spec/index.md` for general specification.

This project itself uses `mem` for the workflow and context management system.

For development, load the `tdd` skill and follow the conventions.
